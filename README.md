# Frontend for WECASTLE game marketplace using Kiosk SDK 
This repo using [SUI Kiosk SDK](https://sdk.mystenlabs.com/kiosk) to build a market place with a simple layout.

User can 
- Place item
- List item for sell
- Take item back to wallet
